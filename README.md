# Stop Slop

Навык для удаления ИИ-шаблонов из прозы.

## Что это такое

У текстов, написанных ИИ, есть паттерны. Предсказуемые фразы, структуры, ритмы. Этот навык учит Клода (или любую большую языковую модель) выявлять и удалять их.

## Структура навыка


```
stop-slop/
├── SKILL.md              # Основные инструкции
├── references/
│   ├── phrases.md        # Фразы для удаления
│   ├── structures.md     # Структурные паттерны, которых стоит избегать
│   └── examples.md       # Преобразования «до/после»
├── README.md
└── LICENSE
```


## Быстрый старт

**Claude Code:** Добавьте эту папку как навык.

**Claude Projects:** Загрузите `SKILL.md` и сопутствующие файлы в базу знаний проекта.

**Пользовательские инструкции:** Скопируйте основные правила из `SKILL.md`.

**API-вызовы:** Включите `SKILL.md` в системный промпт. Справочные файлы подгружаются по требованию.

## Что он выявляет

**Запрещённые фразы** — вступительные «расчистки горла», вспомогательные слова для псевдо-акцентов, бизнес-жаргон, все наречия, размытые утверждения, мета-комментарии. См. `references/phrases.md`.

**Структурные клише** — бинарные противопоставления, перечисления через отрицание, драматическая фрагментация, риторические зачины, ложная олицетворённость, голос повествователя-отстранённого, пассивный залог. См. `references/structures.md`.

**Правила на уровне предложений** — нельзя начинать предложения с вопросительных слов (Wh-), нельзя использовать длинные тире (em dash), нельзя рваную фрагментацию, нельзя ленивые крайности, обязателен активный залог.

Оцените по шкале 1–10 по каждому параметру:

| Параметр | Вопрос |
|----------|--------|
| Прямота | Утверждения или объявления? |

| Доверие | Уважает ли интеллект читателя? |
| Естественность | Звучит как живой человек? |
| Плотность | Есть ли что-то, что можно сократить? |
