# Stop Slop

Навык для удаления ИИ-шаблонов из технических текстов.

## Что это такое

У текстов, написанных ИИ, есть паттерны. Предсказуемые фразы, структуры, ритмы. Этот навык учит языковую модель выявлять и удалять их из технической документации, регламентов, базы знаний и описаний задач.

## Структура навыка


```
stop-slop/
├── SKILL.md              # Основные инструкции
├── references/
│   ├── phrases.md        # Фразы для удаления
│   ├── structures.md     # Структурные паттерны, которых стоит избегать
│   └── examples.md       # Преобразования «до/после»
├── README.md
└── LICENSE
```


## Быстрый старт

**OpenCode:** Поместите все файлы в директорию `.opencode\skills\stop-slop`, так что в этой директории будет лежать SKILL.md и всё остальное

**Claude Code:** Добавьте эту папку как навык.

**Claude Projects:** Загрузите `SKILL.md` и сопутствующие файлы в базу знаний проекта.

**Пользовательские инструкции:** Скопируйте основные правила из `SKILL.md`.

**API-вызовы:** Включите `SKILL.md` в системный промпт. Справочные файлы подгружаются по требованию.

## Что он выявляет

**Запрещённые фразы** — вступительные «расчистки горла», лексические костыли, бизнес-жаргон, все наречия, размытые утверждения, мета-комментарии, пустые лозунги. См. `references/phrases.md`.

**Структурные клише** — бинарные противопоставления, перечисления через отрицание, драматическая фрагментация (в связных текстах), риторические зачины, ложная олицетворённость. См. `references/structures.md`.

**Правила на уровне предложений** — нельзя начинать предложения с вопросительных слов (Что, Когда, Где, Кто, Почему, Как), нельзя ленивые крайности (всегда, никогда, каждый). Активный залог — рекомендуется, но не обязателен.

## Оценка

Оцените по шкале 1–10 по каждому параметру:

| Параметр | Вопрос |
|----------|--------|
| Прямота | Утверждения или объявления? |
| Конкретность | Названы точные значения, условия, сущности? |
| Доверие | Текст написан для людей, которые понимают контекст? |
| Плотность | Есть ли что-то, что можно сократить без потери смысла? |

Если сумма ниже 35/40 — требуется доработка.
